# 🌸 Kottayam Blooms – Website Project Documentation

---

## 📌 Project Overview
This project is a modern, responsive website developed for **Kottayam Blooms** to showcase flower products and allow customers to easily browse items and send enquiries through WhatsApp.

The system is designed for:
- Fast performance ⚡  
- Mobile-first experience 📱  
- Simple product browsing 🛍️  
- WhatsApp-based ordering 💬

## 🛍️ Core Features

### 📦 Product System
- Product listing with images & descriptions
- Categories (Roses, Wedding, Birthday, Gifts, Anniversary, etc.)
- Clean grid-based UI
- Easy to add new products anytime

---

### 📱 WhatsApp Order System
- Direct WhatsApp enquiry button
- Pre-filled order message
- Floating WhatsApp button on all pages
- Main ordering system through WhatsApp (no payment gateway)

---

### 🛒 Enquiry Cart System
- Add / remove products in cart
- Cart summary page
- Local storage support (cart saved in browser)
- Continue shopping option

---

### 🔎 Search & Filter
- Real-time product search
- Category filters
- Sort options (basic UI logic)
- Fast and smooth filtering system

---

### 📱 Responsive Design
- Mobile friendly (fully responsive)
- Tablet & desktop optimized
- Touch-friendly UI components
- Clean and modern layout
